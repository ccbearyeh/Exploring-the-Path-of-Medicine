# 手掌是最近的鏡子——手診快速入門

> **系列**：診斷三部曲 · 第 2 篇 ／ 共 4 篇 ｜ **閱讀時間**：約 4 分鐘

患者一走進診室，張醫師的眼睛就開始工作了。還沒開口問診，已經在看對方的雙手。手診不需要特殊儀器，只需要光線和一雙懂得看的眼睛。

## 手掌是身體最近期的快照

如果說耳診讀的是「過去的歷史」，手診讀的則是**最近這段時間身體的狀態**。手部的顏色、青筋、形態會隨著近期的生活習慣和身體變化而改變，是一個比耳朵更即時的訊號。

張醫師體系的手診從幾個固定區域切入，每個區域對應不同臟腑：

**內關區（PC6，手腕內側）**
- 出現青筋 → 情緒緊張、壓力大
- 三角形區域（女）→ 卵巢囊腫警訊；（男）→ 前列腺問題

**魚際區（LU10，大拇指根部隆起處）**
- 大拇指內側深紅 → 乳房有問題
- 多條橫紋青筋 → 胃寒、消化不良；若同時手掌外側偏紅 → 自律神經失調
- 肉明顯下陷 → 肺虛
- 大拇指腰部細 → 消化不良；粗 → 脂肪肝

**手掌外側**
- 紅色 → 肝功能較弱（對應左關脈）
- 紅腫 → 血糖偏高（對應右關脈）

**指甲**
- 凹陷 → 免疫系統問題、氣血虧損
- 突起 → 肺部積水或末梢循環水腫
- 顏色黑 → 瘀血；白 → 貧血；半透明黏感 → 氣血兩虧

**手肘內側**
- 肺經（LU）處出現青筋或褐色 → 呼吸系統弱
- 心包經（PC）處 → 情緒問題、憂鬱傾向
- 心經（HT）處 → 心臟問題

## 這有什麼臨床意義？

手診的資訊密度很高，一隻手可以同時透露消化、情緒、呼吸、心臟的狀況。它和耳診的搭配邏輯是：**耳診確認有沒有歷史病因，手診確認最近身體在哪裡累積壓力**。初學者可以先從幾個最常見的特徵開始練習：內關青筋（情緒）、魚際橫紋（胃）、指甲顏色（血）。觀察多了，自然會建立起屬於自己的判讀直覺。
