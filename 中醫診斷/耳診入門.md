# 耳廓是倒置的人——認識耳診地圖

> **系列**：診斷三部曲 · 第 1 篇 ／ 共 4 篇 ｜ **閱讀時間**：約 4 分鐘

第一次進診室，張醫師不急著把脈，而是先拿起手電筒，仔細看你的耳朵。這對初學者來說往往有點意外——耳朵跟你說「我腰很痛」有什麼關係？

## 耳廓藏著一張全身地圖

中醫耳診的核心邏輯其實很直觀：**耳廓是一個倒置的嬰兒**。

想像把一個人頭朝下彎曲——耳垂對應頭部，往上走，對耳輪像一條脊椎，三角窩是骨盆，耳甲腔是腹腔臟器，下耳甲腔是胸腔心肺。整張耳廓，就是一份縮小版的人體解剖圖，只是上下顛倒了。

這個概念在中醫裡稱為「體表全息」——某些體表部位是整個人體的縮影，全身任何地方的變化，都會在這張地圖的對應位置留下痕跡。

各區域的對應如下：

| 耳廓區域 | 對應位置 |
|---------|---------|
| 耳垂 | 頭、腦（耳垂有凹槽 → 心臟缺氧警訊） |
| 對耳輪 | 頸椎、脊椎、髖關節 |
| 三角窩 | 婦科、骨盆腔、泌尿系統 |
| 耳甲腔（耳輪腳下） | 胃 |
| 耳甲艇（耳輪腳上） | 腸、肝膽、腎臟 |
| 下耳甲腔 | 心、肺 |
| 對耳屏 | 生殖器官 |
| 耳輪 | 大腸（紅 → 痔瘡；黑點 → 息肉）|

## 顏色就是訊號

知道區域還不夠，更關鍵的是**看顏色**。張醫師體系的判讀規則貫穿整個望診系統：

- **紅色**：當下正在發炎
- **黑色**：曾有瘀血，或受過傷、動過手術後留下的痕跡
- **整個耳輪都紅**：可能是潰瘍性腸炎（自體免疫問題），常伴外表乾瘦、臉色蒼白

舉個例子：對耳輪（脊椎區）發紅，代表頸椎有發炎；這類患者常同時有頭痛、高血壓或腰酸背痛，因為人體五臟六腑都連結著脊椎，脊椎出問題，連帶影響範圍很廣。

## 這有什麼臨床意義？

耳診讀的是「過去的歷史」。黑點告訴你某處曾受傷或有病史，紅色告訴你現在哪裡正在發作。患者或許說不清十年前動過什麼手術，但耳廓上的黑點還在。這就是為什麼初診時，張醫師會先看耳朵——先掌握這個人的身體歷史，再往後理解他現在的狀況。
